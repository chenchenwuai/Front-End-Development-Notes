#### Class和普通构造函数的区别
JS构造函数
```js
function MathHandle(x, y){
    this.x = x;
    this.y = y;
}

MathHandle.prototype.add = function() {
    return this.x + this.y;
}

var m = new MathHandle(1,2)
console.log(m.add())
```

Class基本语法
```js
class MathHandle {
    constructor(x, y){
        this.x = x;
        this.y = y;
    }
    
    add(){
        return this.x + this.y
    }
}

const m = new MathHandle(1,3);
console.log(m.add())
```
